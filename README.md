BlogsApp

A Django-based blog application where users can create blogs and publish posts.

#Features
- Create and manage blogs
- Add, edit, and view blog posts
- User authentication

#Tech Stack
- Python
- Django
- HTML/CSS

#Setup

#Guide
git clone https://github.com/your-username/BlogsApp.git
cd BlogsApp
pip install -r requirements.txt
python manage.py runserver
